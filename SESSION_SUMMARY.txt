DESIGN DISCIPLINE (MAINTAINED THROUGHOUT)
-----------------------------------------

- Zero new runtime deps beyond httpx (already in pyproject). Stripe
  done without the stripe SDK; Turso done without libsql-client.
- @dataclass(frozen=True) from stdlib everywhere. No Pydantic for
  domain types.
- Decimal throughout; .quantize(Decimal('0.000001')) for probabilities,
  string storage for DB round-trips.
- All tests flat in tests/ / tests_api/. No subfolders.
- Append-only migrations -- never edited a shipped migration.
- Every external service has a failsafe path -- every publisher falls
  through to file/email when creds missing; every HTTP client is
  test-mocked via httpx.MockTransport.
